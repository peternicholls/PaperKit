\doxysubsection{Detailed Description}
Represents a color in linear s\+RGB color space. 

Colors are specified as linear RGB components (not gamma-\/corrected). All components are clamped or expected in the range \mbox{[}0, 1\mbox{]}\+:
\begin{DoxyItemize}
\item 0.\+0 = darkest component
\item 1.\+0 = brightest component
\end{DoxyItemize}

Values outside \mbox{[}0, 1\mbox{]} are technically valid (extended RGB) but may be clamped by cj\+\_\+rgb\+\_\+clamp or during conversion to other spaces.

\begin{DoxyNote}{Note}
This is the color format used for input/output throughout the API. 
\end{DoxyNote}
