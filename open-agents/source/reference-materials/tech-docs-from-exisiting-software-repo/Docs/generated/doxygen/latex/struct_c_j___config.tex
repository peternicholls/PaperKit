\doxysubsection{Detailed Description}
Complete configuration for a color journey. 

\doxylink{struct_c_j___config}{CJ\+\_\+\+Config} specifies all parameters that shape a color journey\+: the anchor color(s), perceptual biases, looping behavior, and optional seeded variation. Initialize with cj\+\_\+config\+\_\+init before use.

{\bfseries{Perceptual Biases}}\+:
\begin{DoxyItemize}
\item lightness\+\_\+bias\+: Controls overall brightness
\item chroma\+\_\+bias\+: Controls saturation
\item contrast\+\_\+level\+: Minimum perceptual separation (discrete only)
\item mid\+\_\+journey\+\_\+vibrancy\+: Boost saturation at center of journey \mbox{[}0, 1\mbox{]}
\item temperature\+\_\+bias\+: Shift toward warm or cool hues
\end{DoxyItemize}

{\bfseries{Looping \& Variation}}\+:
\begin{DoxyItemize}
\item loop\+\_\+mode\+: How journey behaves at boundaries (open/closed/pingpong)
\item variation\+\_\+enabled\+: Apply seeded micro-\/variation to discrete colors
\item variation\+\_\+seed\+: Deterministic seed for reproducible variation
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Always initialize with cj\+\_\+config\+\_\+init before modification. 
\end{DoxyNote}
