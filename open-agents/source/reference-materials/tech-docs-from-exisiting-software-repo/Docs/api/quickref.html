üöÄ Swift Quickstart

Basic Palette (5 min)

import ColorJourney let journey = ColorJourney( config: .singleAnchor( ColorJourneyRGB( red: 0.2, green: 0.5, blue: 0.9 ), style: .balanced ) ) let colors = journey.discrete(count: 6)

Start with a single color and get 6 distinct palette colors instantly.

Smooth Gradient

let gradient = journey.linearGradient( stops: 20 ) Rectangle() .fill(gradient) .frame(height: 40)

Create a smooth multi-stop gradient for SwiftUI views.

Custom Config

let config = ColorJourneyConfig( anchors: [baseColor], contrast: .high, chroma: .vivid, lightness: .lighter, loopMode: .closed ) let journey = ColorJourney(config: config)

Full control with explicit parameters for fine-tuning.

Seeded Variation

let journey = ColorJourney( config: ColorJourneyConfig( anchors: [color], variation: .subtle( dimensions: [.hue, .lightness], seed: UInt32(userId) ) ) )

Deterministic per-user color variation using seed.

üîß C Quickstart

Compile & Link

gcc -std=c99 -Wall \ -I./Sources/CColorJourney/include \ main.c \ ./Sources/CColorJourney/ColorJourney.c \ -lm -o myapp

Minimal build command. Link math library (-lm).

Basic Usage

#include "ColorJourney.h" ColorJourneyConfig config = config_init(); ColorJourneyRGB color = RGB(0.2, 0.5, 0.9); ColorJourney journey = create_journey(color, config); ColorJourneyRGB sampled = sample_journey(journey, 0.5); destroy_journey(journey);

Core workflow: init ‚Üí create ‚Üí sample ‚Üí destroy.

Discrete Palette

ColorJourneyRGB* palette = discrete_journey(journey, 8, NULL); for (int i = 0; i < 8; i++) { printf("Color %d: RGB(%.2f, %.2f, %.2f)\n", i, palette[i].red, palette[i].green, palette[i].blue ); } free(palette);

Generate N colors and iterate.

No Dependencies

// Zero external dependencies! // Requires only: // - C99 standard library // - libm (math) // Works on: // - Linux, macOS, Windows // - Embedded systems // - WebAssembly (Emscripten)

Pure C99 + math.h. Extremely portable.

üìã Parameter Cheat Sheet

Style Presets

balanced Default. Medium contrast & chroma. Best for general use.
pastelDrift Soft, desaturated. Good for calm, minimalist designs.
vividLoop High saturation, full spectrum. For data visualization.
nightMode Dark, high contrast. Ideal for dark themes.
warmEarth Warm tones. For natural, earthy palettes.
coolSky Cool tones. For technical, professional designs.

Configuration Options

Parameter Options Default Effect
contrast .low, .medium, .high .medium Color spread in perceptual space
chroma .neutral, .balanced, .vivid .balanced Color saturation level
lightness .lighter, .neutral, .darker .neutral Brightness range
temperature .cool, .neutral, .warm .neutral Color temperature shift
loopMode .open, .closed .open Wrap colors back to start
vibrancy 0.0 to 2.0 1.0 Color intensity multiplier

üìö API Methods Reference

Swift: Sample

journey.sample(at: Float) -> ColorJourneyRGB // Single point sampling let color = journey.sample(at: 0.5)

Get one color at position t ‚àà [0, 1].

Swift: Discrete

journey.discrete(count: Int) -> [ColorJourneyRGB] // N evenly-spaced colors let palette = journey.discrete( count: 12 )

Get N colors spread evenly across journey.

Swift: Gradient

journey.linearGradient(stops: Int) -> LinearGradient // SwiftUI gradient with N stops Rectangle() .fill(journey.linearGradient( stops: 50 ))

SwiftUI LinearGradient ready to use.

C: sample_journey

ColorJourneyRGB sample_journey( ColorJourney j, float t ); // Get color at t ColorJourneyRGB rgb = sample_journey(journey, 0.5);

C equivalent of Swift sample.

C: discrete_journey

ColorJourneyRGB* discrete_journey( ColorJourney j, int count, int* out_count ); // Get N colors ColorJourneyRGB* palette = discrete_journey( journey, 12, NULL );

Returns allocated array. Free with free().

Color Conversion

// RGB to CSS hex let r = UInt8(color.red * 255) let g = UInt8(color.green * 255) let b = UInt8(color.blue * 255) let hex = String( format: "%02X%02X%02X", r, g, b )

Convert to web-friendly formats.

Memory Management

// Swift: Automatic (ARC) let j = ColorJourney(config: config) // Cleaned up automatically // C: Manual cleanup ColorJourney j = create_journey(...); destroy_journey(j);

Swift: automatic. C: call destroy.
