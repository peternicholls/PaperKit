### Swift API Contract
See `swift-api-contract.md`

Formal specification for:
- Public types and functions

- Type safety and discoverability
- Performance characteristics
- Backward compatibility rules

### Documentation Contract
See `documentation-contract.md`

Standards for:
- Comment format and style (Doxygen for C, DocC for Swift)
- Documentation completeness (all public APIs)
- Example code verification
- Terminology consistency
- Reference standards and tools
- Maintenance procedures
