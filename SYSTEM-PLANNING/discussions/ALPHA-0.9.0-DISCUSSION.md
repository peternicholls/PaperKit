## How to Test

1. **Clone the repository**:
   ```bash
   git clone https://github.com/peternicholls/PaperKit.git
   cd PaperKit
   ```

2. **Set up your environment**:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r open-agents/tools/requirements.txt
   ```

3. **Start writing a paper**:
   - Open GitHub Copilot Chat in VS Code
   - Select the `paper-architect` agent
   - Try: "Create an outline for my paper on [your topic]"

4. **Explore the system**:
   - Check [AGENTS.md](https://github.com/peternicholls/PaperKit/blob/master/AGENTS.md) for all commands
   - See [README.md](https://github.com/peternicholls/PaperKit/blob/master/README.md) for workflow overview
